# Changelog 2026-04

- Copper-Pour workflow overhaul
  - Add `unbroken` flag to pours [props#628](https://github.com/tscircuit/props/pull/628) → core support [core#2129](https://github.com/tscircuit/core/pull/2129)
  - Autorouter gains escape-via stage & via-clearance DRC ([autorouter#875](https://github.com/tscircuit/tscircuit-autorouter/pull/875), [#917](https://github.com/tscircuit/tscircuit-autorouter/pull/917))
  - Inner-layer & translucent pour rendering in canvas / viewer ([canvas#228](https://github.com/tscircuit/circuit-to-canvas/pull/228), [pcb-viewer#737](https://github.com/tscircuit/pcb-viewer/pull/737))
  - Solver now treats pill/rect plated holes as clearance obstacles [copper-pour-solver#36](https://github.com/tscircuit/copper-pour-solver/pull/36)

- Footprint reliability improvements

  - Correct dimensions for SOT/SOD/axial families [footprinter#593](https://github.com/tscircuit/footprinter/pull/593)

- New design-rule checks & schema
  - Pad-pad / pad-trace and via-trace clearance errors now generated & rendered ([checks#134](https://github.com/tscircuit/checks/pull/134), [#135](https://github.com/tscircuit/checks/pull/135), [svg#546](https://github.com/tscircuit/circuit-to-svg/pull/546))

  - CLI gains `tsci check trace-length` [cli#2633](https://github.com/tscircuit/cli/pull/2633)

- PCB Viewer UX upgrades
  - Group/collapse error list [pcb-viewer#733](https://github.com/tscircuit/pcb-viewer/pull/733)
  - Smooth zoom to first error & diamond markers [pcb-viewer#741](https://github.com/tscircuit/pcb-viewer/pull/741), [#745](https://github.com/tscircuit/pcb-viewer/pull/745)
  - Disable canvas when using toolbar overlays [pcb-viewer#729](https://github.com/tscircuit/pcb-viewer/pull/729)
