# Changelog 2026-04

- Copper pours can now be declared **“unbroken”** to keep contiguous copper islands intact ([props #628](https://github.com/tscircuit/props/pull/628), [core #2129](https://github.com/tscircuit/core/pull/2129))
- Autorouter gains escape-via stage, proper via-to-copper clearance and high-density force-improve solver for tougher boards ([autorouter #875](https://github.com/tscircuit/tscircuit-autorouter/pull/875), [#854](https://github.com/tscircuit/tscircuit-autorouter/pull/854), [#876](https://github.com/tscircuit/tscircuit-autorouter/pull/876))
- Footprinter switches to generic body-based courtyards and fixes courtyard/dimension errors across dozens of packages (e.g. [footprinter #585](https://github.com/tscircuit/footprinter/pull/585), [#590](https://github.com/tscircuit/footprinter/pull/590), [#593](https://github.com/tscircuit/footprinter/pull/593))
- Inner-layer copper pours now render correctly and support translucent colours in canvas viewer ([pcb-viewer #737](https://github.com/tscircuit/pcb-viewer/pull/737), [circuit-to-canvas #228](https://github.com/tscircuit/circuit-to-canvas/pull/228))
- PCB courtyard outlines now use rounded line-joins in SVG/Canvas outputs ([circuit-to-canvas #220](https://github.com/tscircuit/circuit-to-canvas/pull/220), [circuit-to-svg #538](https://github.com/tscircuit/circuit-to-svg/pull/538))
- EasyEDA importer improvements: keeps {NAME} placeholders, corrects CAD model placement and supports oval/pill plated holes ([easyeda-converter #374](https://github.com/tscircuit/easyeda-converter/pull/374), [#375](https://github.com/tscircuit/easyeda-converter/pull/375), [#377](https://github.com/tscircuit/easyeda-converter/pull/377))
- KiCad importer gains inner-layer support and accurate oval-hole drill/pad rotations ([kicad-to-circuit-json #54](https://github.com/tscircuit/kicad-to-circuit-json/pull/54), [#58](https://github.com/tscircuit/kicad-to-circuit-json/pull/58), [#60](https://github.com/tscircuit/kicad-to-circuit-json/pull/60))
- USB-C connectors: added inner schematic symbol and eliminated false missing-footprint errors ([core #2118](https://github.com/tscircuit/core/pull/2118), [#2136](https://github.com/tscircuit/core/pull/2136))
- Schematic symbols now respect new _horz / _vert orientation suffixes ([core #2138](https://github.com/tscircuit/core/pull/2138))
- PCB Viewer UX: blocks toolbar-overlay clicks, groups error list, keeps drill layer on top and highlights via-clearance errors ([pcb-viewer #729](https://github.com/tscircuit/pcb-viewer/pull/729), [#733](https://github.com/tscircuit/pcb-viewer/pull/733), [#735](https://github.com/tscircuit/pcb-viewer/pull/735), [#738](https://github.com/tscircuit/pcb-viewer/pull/738))
- CLI adds `tsci check trace-length` and configurable worker timeout ([cli #2633](https://github.com/tscircuit/cli/pull/2633), [#2667](https://github.com/tscircuit/cli/pull/2667))
- CAD component viewer gets drag-and-drop landing page, preserves model hierarchy and material colours ([cad-component-viz #5](https://github.com/tscircuit/cad-component-viz/pull/5), [#7](https://github.com/tscircuit/cad-component-viz/pull/7))
- STEP exporter now outputs coloured faces and reuses styles for smaller files ([circuit-json-to-step #71](https://github.com/tscircuit/circuit-json-to-step/pull/71))
- KiCad export fixes netlabel anchor-side mapping for deterministic label placement ([circuit-json-to-kicad #200](https://github.com/tscircuit/circuit-json-to-kicad/pull/200))
- Missing KiCad 3D models now fall back to corresponding STEP files ([kicad-mod-cache #21](https://github.com/tscircuit/kicad-mod-cache/pull/21))
