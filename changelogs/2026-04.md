# Changelog 2026-04

- Added support for “unbroken” copper pours across the stack – design-time flag, rendering and routing awareness ([props/#628](https://github.com/tscircuit/props/pull/628), [core/#2129](https://github.com/tscircuit/core/pull/2129), [autorouter/#886](https://github.com/tscircuit/tscircuit-autorouter/pull/886))
- New escape-via stage connects signals to pours and improves scoring/DRC for vias ([autorouter/#875](https://github.com/tscircuit/tscircuit-autorouter/pull/875), [autorouter/#880](https://github.com/tscircuit/tscircuit-autorouter/pull/880), [autorouter/#917](https://github.com/tscircuit/tscircuit-autorouter/pull/917))
- Phased local autorouting enabled via routingPhaseIndex on nets & traces ([props/#629](https://github.com/tscircuit/props/pull/629), [core/#2139](https://github.com/tscircuit/core/pull/2139))
- Massive courtyard overhaul: all SO**, TSSOP/SOP/SON/VSSOP/BGA/DIP/AXIAL/quad packages now use body-based stepped outlines and KiCad-dimension fixes ([footprinter/#584](https://github.com/tscircuit/footprinter/pull/584)–[footprinter/#594](https://github.com/tscircuit/footprinter/pull/594))
- EasyEDA import fixes – preserved {NAME} placeholders, accurate CAD placement & pill-hole support ([easyeda-converter/#374](https://github.com/tscircuit/easyeda-converter/pull/374), [#375](https://github.com/tscircuit/easyeda-converter/pull/375), [#377](https://github.com/tscircuit/easyeda-converter/pull/377))
- PCB Viewer UX improvements: toolbar no-click-through, error grouping & smooth zoom-to-error, inner-layer pour rendering and drill layer priority ([viewer/#729](https://github.com/tscircuit/pcb-viewer/pull/729), [#733](https://github.com/tscircuit/pcb-viewer/pull/733), [#737](https://github.com/tscircuit/pcb-viewer/pull/737), [#741](https://github.com/tscircuit/pcb-viewer/pull/741), [#735](https://github.com/tscircuit/pcb-viewer/pull/735))
- circuit-to-canvas now supports round line-joins, translucent inner-layer pours and auto-mirrors bottom-copper text ([canvas/#220](https://github.com/tscircuit/circuit-to-canvas/pull/220), [#228](https://github.com/tscircuit/circuit-to-canvas/pull/228), [#226](https://github.com/tscircuit/circuit-to-canvas/pull/226))
- CAD Component Viewer: drag-and-drop landing page, full UI redesign with loading states and Three.js optimizations ([cad-component-viz/#5](https://github.com/tscircuit/cad-component-viz/pull/5), [#9](https://github.com/tscircuit/cad-component-viz/pull/9))
- STEP/GLTF export pipeline gains per-face color styling and higher-res board textures ([cj-to-step/#71](https://github.com/tscircuit/circuit-json-to-step/pull/71), [runframe/#3049](https://github.com/tscircuit/runframe/pull/3049))
- New CLI features: `tsci check trace-length` and configurable worker timeouts ([cli/#2633](https://github.com/tscircuit/cli/pull/2633), [#2667](https://github.com/tscircuit/cli/pull/2667))
- DRC/Checks: thickness-mismatch warnings & different-net via spacing enforcement ([checks/#132](https://github.com/tscircuit/checks/pull/132), [autorouter/#917](https://github.com/tscircuit/tscircuit-autorouter/pull/917))
- Chips can now mark pins as `noConnect`, suppressing stray-trace warnings ([props/#632](https://github.com/tscircuit/props/pull/632), [core/#2147](https://github.com/tscircuit/core/pull/2147))
- USB-C schematic symbol improvements and inner-symbol support ([core/#2118](https://github.com/tscircuit/core/pull/2118), [#2131](https://github.com/tscircuit/core/pull/2131))
- Kicad import/export: inner-layer support, arcs & pill-hole alignment, accurate via sizes ([kicad-to-cj/#54](https://github.com/tscircuit/kicad-to-circuit-json/pull/54), [#58](https://github.com/tscircuit/kicad-to-circuit-json/pull/58), [cj-to-kicad/#205](https://github.com/tscircuit/circuit-json-to-kicad/pull/205))
